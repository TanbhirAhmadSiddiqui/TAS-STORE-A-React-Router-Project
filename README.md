🚀 App Development Website

A modern App Development Website built using React, React Router, Tailwind CSS, and Vite. This project demonstrates real-world frontend concepts like routing, data fetching, local storage management, sorting, and error handling.

📌 Live Demo

https://tas-store-a-react-router-project.vercel.app/

🛠️ Technologies Used
⚛️ React
🔀 React Router
⚡ Vite
🎨 Tailwind CSS
🌼 DaisyUI
📊 Recharts
🗂️ Fake JSON Data (for API simulation)
💾 Local Storage
✨ Features
📄 Dynamic Routing
Multi-page navigation using React Router
📊 Data Visualization
Charts built with Recharts
🔄 Data Fetching
Fake JSON data used to simulate API calls
💾 Local Storage Integration
Save and retrieve user-selected data
🔍 Sorting Functionality
Sort data based on size or other parameters
⏳ Loading States
Smooth UI experience with loading indicators
❌ Error Handling
Custom error pages (404, etc.)
Graceful fallback UI
📱 Responsive Design
Fully responsive with Tailwind CSS & DaisyUI
